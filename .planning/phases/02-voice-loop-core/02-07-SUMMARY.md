## Human Action Required

The following steps require human action and cannot be automated:

### Step 1: Configure Git Remote and Push to Railway

No git remote is configured in this repository. Railway deployment requires either:
- **Option A (GitHub Integration):** Push the repo to GitHub, then connect Railway to the GitHub repo in the Railway dashboard
- **Option B (Railway CLI):** Install Railway CLI (`npm i -g @railway/cli`), run `railway login`, `railway link`, then `railway up`

Once the remote is configured:
```bash
git push origin main
# OR: railway up (if using Railway CLI directly)
```

### Step 2: Set Environment Variables in Railway Dashboard

In Railway Dashboard → your JARVIS project → Variables tab:
- `DEEPGRAM_API_KEY` — from console.deepgram.com → API Keys → Create Key (member role)
- `CLAUDE_API_KEY` — from console.anthropic.com → API Keys
- `PORT` — set to `8080` (required for uvicorn CMD)
- `MONGO_URL` — auto-injected by Railway MongoDB plugin (verify it's present)

After setting variables, trigger a redeploy.

### Step 3: Smoke Test

```bash
# Replace YOUR-RAILWAY-URL with your actual Railway domain
curl -s https://YOUR-RAILWAY-URL.railway.app/api/health
# Expected: {"status": "ok", "mongo": "connected"}

curl -s -X POST https://YOUR-RAILWAY-URL.railway.app/api/chat \
  -H "Content-Type: application/json" \
  -d '{"transcript": "hello", "session_id": "smoke-test-001"}'
# Expected: {"mode": "speak", "text": "...", "fetch": "none", "query": ""}
```

### Step 4: iPad End-to-End Validation

On iPad in standalone PWA mode (added to home screen):

**Test 1: English voice loop (baseline)**
1. Open JARVIS PWA from iPad home screen
2. Tap the orb/screen → say "What is the capital of France?"
3. Verify: Blue waveform reacts to voice (ListeningMode)
4. Verify: After 1.5-2s silence → ThinkingMode orb appears
5. Verify: SpeakingMode with purple waveform + subtitle text
6. Verify: JARVIS says "Paris" audibly (TTS)
7. Verify: Returns to idle after speaking

**Test 2: Russian language**
1. Tap → say in Russian: "Какая столица Франции?"
2. Verify: Response in Russian (CONV-02)

**Test 3: Conversation history**
1. Ask: "Tell me about Paris"
2. After response, ask: "What is its most famous landmark?"
3. Verify: JARVIS responds about Paris's landmark (context carry-over)

**Test 4: Tap to stop**
1. Ask a long question → while JARVIS speaks, tap the screen
2. Verify: Speech stops immediately → returns to idle (TTS-03)

**Test 5: Silence detection timing**
1. Tap → say "Um..." then pause
2. Verify: Recording stops after ~1.5-2s (VOICE-04)
